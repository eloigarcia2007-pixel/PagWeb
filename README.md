# Tallers Egasa — Pàgina web

Projecte estàtic senzill per a un taller mecànic, pensat per desenvolupar amb Visual Studio Code.

Fitxers creats:
- `index.html` — Estructura principal
- `css/styles.css` — Estils separats
- `js/main.js` — JavaScript bàsic (menú i formulari)

Com provar-ho localment:
1. Obre la carpeta del projecte amb Visual Studio Code.
2. Pots obrir `index.html` directament amb el navegador o utilitzar una extensió com "Live Server" per recàrrega en viu.

Comandes d'exemple (Windows PowerShell):

```powershell
# Obre el fitxer index.html al navegador per defecte
start .\index.html
```

Extensió recomanada: `ritwickdey.LiveServer` per a recàrrega en calent mentre desenvolupes.

Notes:
- El formulari és de prova; per enviar realment dades cal integrar una API o un servei de servidor.
- El disseny és responsive i es pot ampliar fàcilment.
